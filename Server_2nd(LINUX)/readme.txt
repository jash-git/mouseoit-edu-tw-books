附錄D 光碟檔案使用說明
    
如果您使用光碟機為IDE介面且為第二部，則可利用

    mount  -t  iso9660  /dev/hdb  /cdrom

    將光碟檔案掛在 /cdrom 目錄下，注意如果你看到不是長檔名的話，如 apache~8.gz，那請你使用apache~8.gz 來作本書的練習。
